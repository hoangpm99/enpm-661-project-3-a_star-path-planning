Dependencies
--------------------------------------------------------------------------------
To run this project, you need the following dependencies:

Python 3.x
OpenCV (cv2)
numpy
math
time

--------------------------------------------------------------------------------
Running the Code
--------------------------------------------------------------------------------
Clone or download the repository to your local machine.

```
https://github.com/hoangpm99/enpm-661-project-3-a_star-path-planning.git
```

Ensure you have Python and OpenCV installed.
Navigate to the project directory in your terminal or command prompt.
Run the following command to execute the program:

```
python a_star_hoang_fazil.py
```

You will be prompted to enter the start and goal coordinates and orientation for the path planning. 

Enter in the format (x y theta) and press Enter for both Start and Goal


*Another option is open file in VSCode and use CodeRunner.
